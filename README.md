# POS Inventory Management System

A modern POS (Point of Sale) and Inventory Management System built with Django.  
This system helps businesses manage products, sales, inventory, invoices, customers, and reports efficiently.

---

## 🚀 Features

- User Authentication & Role Management
- Product & Category Management
- Inventory Tracking
- POS Billing System
- Invoice Generation
- Customer Management
- Supplier Management
- Sales & Purchase Reports
- Dashboard Analytics
- Responsive Design

---

## 🛠 Tech Stack

### Backend
- Python
- Django

### Frontend
- HTML
- CSS
- JavaScript
- Bootstrap

### Database
- SQLite
